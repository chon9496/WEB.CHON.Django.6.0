# Ambiente:

## Instalar:

    pip install virtualenv env

## Crear:

    virtualenv env
                     
## Activar:

    cd env/Scripts
    activate
    cd../..

### Activar en Linux y mac:

    source env/bin/activate

## requirements.txt:
Creamos un archivo con el nombre requirements.txt

Ejecutamos en la terminal

    pip install -r requirements.txt
